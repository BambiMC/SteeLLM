# Requirements

- CMake (optional)
- Python
- Pip
- Git
- Linux

# How to run SteeLLM

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Fill in the parameters in the config.json file

   ```bash
   nano config.json
   ```

3. Run the official benchmark from within the scripts directory:
   ```bash
   cd scripts
   bash official_benchmark.sh
   ```
