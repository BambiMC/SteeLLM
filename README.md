# Requirements

- CMake (optional)
- Python
- Pip
- Git
- Linux

# How to run SteeLLM

1. Clone the repository:

   ```bash
   git clone <repository-url>
   ```

2. Fill in the api_keys.json & environment.json file with your API keys.

   ```bash
   nano api_keys.json
   nano environment.json
   ```

3. Run the official benchmark:
   ```bash
   cd scripts
   bash official_benchmark.sh
   ```
